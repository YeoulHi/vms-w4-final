- **파일 경로**: {구현할 파일의 전체 경로}
- **담당 모듈**: {해당 파일이 담당하는 모듈 이름}

**규칙:**
1.  plan.md에 명시된 인터페이스와 로직을 정확히 구현해야 한다.
2.  다른 파일이나 모듈의 구현은 절대 포함하지 말고, 오직 명시된 파일의 코드만 작성하라. (필요시 import 구문은 사용 가능)
3.  type, lint, build 에러가 없어야 한다.
4.  주석을 통해 코드의 의도를 간략히 설명하라.
